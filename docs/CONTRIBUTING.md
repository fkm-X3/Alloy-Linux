# Contributing

Thank you for contributing to Alloy-Linux. Please follow packaging and repository conventions:

- Put package recipes in packages/(name)/ with an Alloyfile and build.sh
- Keep build scripts in build/ and document choices in build/README.md
- Prefer Rust for new project-owned tooling/runtime components
- Keep target/profile metadata in `meta/`
- Open PRs for patches and follow code review guidelines in docs/
