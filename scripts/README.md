Scripts

Developer helper scripts (mkimage, sign, packaging helpers).

- `toolchain/setup-toolchain.sh`: validates ARM64 cross toolchain and emits `build/output/toolchain.env`.
- `toolchain/setup-toolchain.ps1`: Windows PowerShell toolchain validator that emits `build/output/toolchain.env.ps1`.
- `toolchain/setup-host.ps1`: Windows host diagnostics/bootstrap wrapper for toolchain setup.
