Installer

Text-based installer prototype and scripts. Installer should create partitions, install base system, and configure bootloader.

Primary install target is Orange Pi 5 (RK3588) with a U-Boot boot flow.

Current status:
- `installer/install.sh` is a scaffold and does not execute partitioning/formatting yet.
- See `docs/windows-flash-install.md` for current Windows flashing and manual bridge steps.
